📋 How to use

  • Open this encrypt.html file in your browser on your desktop (just double-click it — no internet needed)
  • Drop your MP3 files into the box above and click Encrypt & Download All
  • Each file downloads as filename.ganuman — save them anywhere
  • Upload the .ganuman files into your GitHub repo's Database/ folder
  • In MusicPro → Settings → Your Tracks, list each file with .ganuman extension (e.g. mysong.ganuman)
  • The player will automatically decrypt and stream each file when you play it

🛡 Security details

Each file is protected with:

  • AES-256-GCM — military-grade symmetric encryption. The same algorithm used by banks and governments.
  • PBKDF2 with 200,000 iterations — even a simple password becomes computationally expensive to brute-force
  • Random salt + IV per file — encrypting the same file twice produces completely different output, preventing pattern analysis
  • Custom .ganuman magic header — anyone who finds your files won't even know what format they are
  • 100% browser-side — your files and password never leave your device during encryption

⚠️ Keep this file private. The encryption key is embedded inside encrypt.html and index.html. Do not share or publish these files. If you push them to a public GitHub repo, anyone who reads the source code will find the key. Host your player on GitHub Pages but keep the source repo private, or accept that determined people could find the key in the HTML source.
